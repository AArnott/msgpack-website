Description

MessagePack codec for Erlang.

APIs are almost compatible with C API except for buffering functions (both copying and zero-copying), which are unavailable.

pack/1

pack(Term::msgpack_term()) -> binary() | {error, {badarg, term()}}

Encode an erlang term into an msgpack binary. Returns {error, {badarg, term()}} if the input is illegal.

unpack/1

unpack(Bin::binary()) -> {msgpack_term(), binary()} | {error, incomplete} | {error, {badarg, term()}}

Decode an msgpack binary into an erlang term. It only decodes the first msgpack packet contained in the binary; the rest is returned as is. Returns {error, {badarg, term()}} if the input is corrupted. Returns {error, incomplete} if the input is not a full msgpack packet (caller should gather more data and try again).

unpack_all/1

unpack_all(Data::binary()) -> [msgpack_term()] | {error, incomplete} | {error, {badarg, term()}}

Decode an msgpack binary into an erlang terms. It only decodes ALL msgpack packets contained in the binary. No packets should not remain. Returns {error, {badarg, term()}} if the input is corrupted. Returns {error, incomplete} if the input is not a full msgpack packet (caller should gather more data and try again).
